Розрахунок військового збору для фізичних осіб

Ставка військового збору також стандартна завжди – 1,5% від бази оподаткування. Базою оподаткування військовим збором є сам дохід. Формула для розрахунка така:

ВЗ = Зарплата (винагорода за ЦПД, інший об’єкт оподаткування) × 0,015

Розрахунок військового збору для фізичних осіб-підприємців

Для фізичних осіб-підприємців на загальній системі військовий збір рахується від їхнього чистого оподатковуваного доходу, що є різницею між доходами та дозволеними витратами:

ВЗ для ФОП = Чистий оподатковуваний дохід × 0,015
